# IS2 ETF DATA WAREHOUSE DESIGN

- Complete assignment you can check in the 13M111IS2_domaci_22_23 file.

- Tech stack requirements
    * MySQL Server
    * MySQL Workbench
    * Kettle (Spoon)
    * Mondrian Schema Workbench
- Steps:
  * Create operational database and populate it with the data
  * Create data warehouse schema
  * Create Kattle (Spoon) files for warehouse increment loading
  * Create MDX queries
